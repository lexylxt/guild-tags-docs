# API

GuildTags does not currently offer a public API.

---

## Why no public API?

The GuildTags platform is focused on providing a smooth experience through its website and Discord bot. Exposing a public API introduces significant challenges around abuse prevention, rate limiting, and data privacy — particularly around server invite links, which are protected by the secure redirect system.

To keep the directory reliable and free from scraping, we have made the deliberate decision not to publish a public API at this time.

---

## Future plans

A limited read-only API may be considered in the future for verified partners and developers building Discord-related tools. There are no confirmed plans or timelines for this.

If you have a specific use case that would benefit from API access, you are welcome to reach out via the [GuildTags Discord server](https://discord.com/invite/9CqvbP6Yfx) to discuss it with the team.

---

## Alternatives

If you need tag data for a specific purpose:

- The [directory](/documentation/en/web/dashboard) is publicly accessible to logged-in users
- The bot can be used to query your own server's status via [/status](/documentation/en/bot/commands)
- For partnership or data sharing requests, contact us on Discord
