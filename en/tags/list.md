# Tag list

The GuildTags directory displays all registered server tags in one place. This page explains how the listing works, how to search it, and what each piece of information means.

---

## Accessing the directory

The full tag directory is available at [guildtags.com/tags](https://guildtags.com/tags). You must be logged in with your Discord account to view the listings. Login is required to prevent automated scraping of server invite links.

---

## Directory columns

Each row in the directory represents one registered server and shows the following information:

| Column | Description |
|---|---|
| Badge | A verified or featured badge if applicable |
| Tag | The server's active Discord clan tag |
| Server | The server name and icon |
| Join | A secure button to join the server |
| Server ID | The Discord server ID |
| Likes | The total number of likes from GuildTags users |

---

## Searching and filtering

The directory includes a search bar at the top of the page. You can search by:

- **Tag** — enter the exact tag or a partial match
- **Server name** — search by the name of the Discord server
- **Server ID** — paste a Discord server ID to find an exact match

Results update as you type. The search is not case-sensitive.

---

## Sorting

By default, the directory sorts listings by the time they were added, with the most recent at the top. You can also sort by:

- **Most liked** — servers with the highest like count appear first
- **Alphabetical** — sorted by tag characters from A to Z

Featured servers are always shown at the top regardless of sort order.

---

## Badge types

**Verified** — the server has been reviewed by the GuildTags team and confirmed as a genuine community.

**Featured** — the server has an active Featured plan subscription and is pinned at the top of the directory.

---

## Joining a server

Clicking the Join button on any listing will route you through the GuildTags redirect system before landing on the Discord invite page. This protects invite links from being harvested by bots. The redirect is instantaneous and you will see a brief intermediary page before being forwarded.
