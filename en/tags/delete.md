# Deleting a tag

If you want to remove your server from the GuildTags directory, you can unregister it at any time. This will remove the listing and all associated data from the public directory.

---

## Unregistering your server

Use the following command in your Discord server to remove your listing:

```
/unregister
```

The bot will ask for confirmation before proceeding. Once confirmed, your server's tag listing will be removed from the directory immediately.

> [!WARNING]
> Unregistering is permanent. All likes your listing had accumulated will be lost. If you re-register later, the like count will start from zero.

---

## What gets deleted

When you unregister, the following data is removed from the GuildTags database:

- Your clan tag listing
- Your server's invite link association
- All likes on your listing
- Your position on the leaderboard

Your Discord server itself is not affected in any way — only the GuildTags directory entry is removed.

---

## Requesting data deletion

If you want to request a complete deletion of all data associated with your server (including any analytics or audit records), contact the GuildTags team via the [Discord server](https://discord.com/invite/9CqvbP6Yfx) or submit a request through the Privacy section of the website. Requests are processed within 48 hours.

---

## Removing the bot

If you also want to remove the GuildTags bot from your server, go to your Discord server settings, navigate to Integrations, find the GuildTags bot, and click Remove. This will not automatically unregister your listing — you should run `/unregister` before removing the bot if you want to remove your directory entry.
