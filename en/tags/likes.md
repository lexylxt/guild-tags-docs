# Likes and leaderboard

GuildTags includes a like system that lets users express support for their favourite servers. Likes determine rankings on the public leaderboard.

---

## Liking a server

Any logged-in user can like a server from the directory. To like a listing, click the heart or like button on any server card in the [directory](https://guildtags.com/tags). Each user can like a given server once. You can remove your like at any time by clicking the button again.

> [!NOTE]
> Liking requires a GuildTags account linked to Discord. If you are not logged in, you will be prompted to connect your Discord account first.

---

## The leaderboard

The leaderboard at [guildtags.com/leaderboard](https://guildtags.com/leaderboard) ranks all registered servers by their total number of likes. It is updated in real time as likes are cast and removed.

The leaderboard shows two tabs:

**Most Liked** — servers ranked by total likes received across all time.

**Most Voted** — servers ranked by recent engagement (likes received in the current period).

The top three servers are displayed on a podium. Servers ranked 4th and beyond are shown in a scrollable list below.

---

## How rankings work

Rankings are calculated from the total number of unique likes each server has received. There is no time decay — older likes count equally to new ones on the Most Liked tab. The Most Voted tab uses a rolling window to highlight recently active servers.

Featured servers are not pinned to the top of the leaderboard — they compete fairly based on their like count.

---

## Preventing abuse

Each Discord account can like a given server only once. Attempts to inflate like counts through multiple accounts or automation violate GuildTags' terms of service and may result in the server being removed from the directory.
