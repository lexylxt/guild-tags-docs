# What is a tag?

A Discord clan tag is a short label — usually two to five characters — that Discord displays next to a server member's username when they are active in that server. Tags were introduced by Discord as part of the clan system, allowing communities to express a shared identity through a visible prefix or suffix.

---

## How tags work on Discord

When a server enables a clan tag, members who belong to that clan see the tag appear beside their display name in the member list, in chat messages, and on their profile within that server. The tag is set by the server's administrators and is unique to that server's identity.

Tags are separate from usernames. A user called `alex` with the clan tag `GT` would appear as `[GT] alex` or `alex · GT` depending on Discord's display format.

> [!NOTE]
> Clan tags are a Discord feature, not a GuildTags feature. GuildTags indexes and displays existing Discord clan tags — it does not create or modify tags directly on Discord.

---

## What GuildTags does with tags

GuildTags collects tag information from registered Discord servers and displays them in a public directory at [guildtags.com](https://guildtags.com). This makes it easy for anyone to:

- Search for servers by their tag
- Browse communities and discover new ones
- Like favourite servers and track them on the leaderboard
- Join a server directly from the directory listing

Each tag listing in the directory shows the tag itself, the server name, a badge (if the server has been verified or featured), the number of likes, and a secure join link.

---

## Tag requirements

For a tag to appear in the GuildTags directory, the server must:

- Have an active Discord clan tag set through Discord's server settings
- Have the GuildTags bot added to the server
- Be registered by a server administrator through the bot or the website

Tags must comply with Discord's community guidelines and GuildTags' terms of service. Tags that are offensive, misleading, or in violation of either policy will be removed.
